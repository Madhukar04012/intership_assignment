## ðŸ“‹ Assignment Details

### Task 1: Python Basics & Operators
- Takes two numbers as input.
- Performs addition, subtraction, multiplication.
- Implements safe division and modulus (handles division by zero).
- Demonstrates comparison and logical operators.

### Task 2: Lists & NumPy
- Generates a list of 10 random integers.
- Safely removes elements using `pop()`.
- Sorts the list and finds max/min values.
- Converts the list to a **NumPy array** to calculate:
    - Mean
    - Median
    - Standard Deviation

### Task 3: Dictionaries & Sets
- **Dictionary**: Stores student name, course, and marks. automatically assigns a grade (A/B/C/D) based on marks.
- **Sets**: Manages two sets of AI tools to demonstrate `union` and `intersection` operations.

### Task 4: File Handling
- Creates a file `ai_students.txt` with student data.
- Reads the file and filters/displays students who scored **above 75 marks**.

### Task 5: Real-World Mini Project (AI Prompt Logger)
- Simulates an AI tool's logging system.
- Accepts a user prompt, attaches the current **timestamp**, and appends it to `prompt_history.txt`.
